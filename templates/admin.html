Admin Dashboard Guide

1. Review Feedback

Check the user query and the bot's response. Look for inaccuracies or hallucinations.

2. Edit Response

If the answer is incorrect, edit the text in the "Bot Response" box to provide the correct information.

3. Save to Golden Dataset

Click "Save" to store the verified Q&A pair. This builds a high-quality dataset for future training.

4. Ingest to Search Index

Click "Ingest" to immediately add the verified Q&A pair to the CloudProvider Search index, making it available for future queries.
